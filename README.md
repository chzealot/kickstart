# kickstart

一键初始化新电脑环境的命令行工具。

- 管理 dotfiles
- 安装常用工具和软件包
- 配置软件和系统偏好设置

## 安装

**Mac / Linux：**

```bash
curl -fsSL https://raw.githubusercontent.com/chzealot/kickstart/main/install.sh | bash
```

**Windows：**

```powershell
irm https://raw.githubusercontent.com/chzealot/kickstart/main/install.ps1 | iex
```
